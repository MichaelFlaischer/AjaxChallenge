My Address Book Challenge with XMLHttpRequest

In this section of the AJAX Challenge, I created a dynamic address book. The task was to fetch and display contact information using `XMLHttpRequest`. I implemented this by making raw AJAX requests to retrieve data from a server and then dynamically rendering that data on the page. This approach allowed me to enhance my understanding of handling asynchronous data fetching and manipulating the DOM to display the fetched information efficiently.
