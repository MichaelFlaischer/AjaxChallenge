Welcome to the Movie Genres Challenge

Implementing a Movie Genres and Details Viewer

In this part of the AJAX Challenge, I built a dynamic movie genre viewer that fetches and displays genres using XMLHttpRequest. When a genre is selected, the application fetches and displays corresponding movie details, including the title, overview, and an image. The images are loaded according to the guidelines provided by The Movie Database (TMDb) API to ensure a smooth user experience. To avoid exceeding the API's call limits, data is cached locally in the browser.
